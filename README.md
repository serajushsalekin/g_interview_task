# g_interview_task
## Installation

Install [meteor](https://www.meteor.com/install) .

```bash
git clone git@github.com:serajushsalekin/g_interview_task.git
```
```
meteor npm install
```

## Usage
### run app
```
meteor
```
### mongo shell
```
meteor mongo
```
